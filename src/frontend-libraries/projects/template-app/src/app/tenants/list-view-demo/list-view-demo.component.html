mm-list-view Documentation

A powerful data grid wrapper based on Kendo UI Grid with additional features.

Features:

  • Sortable: Click column headers to sort
  • Row Filter: Per-column filtering via toolbar
  • Search: Full-text search with debounce
  • Selection: Single/multiple row selection
  • Context Menu: Right-click or action button
  • Row Actions: Per-row action buttons
  • Toolbar: Custom left/right actions
  • Export: Excel and PDF export
  • Pagination: Configurable page sizes
  • Column Types: text, numeric, boolean, date, iso8601, bytes, statusIcons

API Reference
Property Type Default Description
[columns] ColumnDefinition[] [] Column definitions with field, displayName, dataType, format
[sortable] boolean false Enable column sorting
[rowFilterEnabled] boolean false Enable row-level filtering
[searchTextBoxEnabled] boolean false Show search text box in toolbar
[selectable] SelectableSettings {{ '{enabled: false}' }} Configure selection mode and enabled state
[contextMenuType] 'contextMenu' | 'actionMenu' 'actionMenu' How to trigger context menu
[actionCommandItems] CommandItem[] [] Action buttons in each row
[contextMenuCommandItems] CommandItem[] [] Context menu items
[leftToolbarActions] CommandItem[] [] Left toolbar buttons
[rightToolbarActions] CommandItem[] [] Right toolbar buttons
[pageSize] number 10 Rows per page
[excelExportFileName] string 'Export.xlsx' Excel export filename
[pdfExportFileName] string 'Export.pdf' PDF export filename
Column DataTypes
  • text - Standard text
  • numeric - Number formatting
  • boolean - Checkbox display
  • date - Native date
  • iso8601 - ISO date strings
  • bytes - File sizes (human-readable)
  • statusIcons - Enum values as colored icons with tooltips
